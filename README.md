## ✨ Overview

This portfolio is more than a personal website—it's an interactive playground showcasing my work, frontend engineering skills, animations, and creative experiments.

Designed with a cinematic aesthetic, it combines smooth motion, premium UI, and performance-focused architecture to create a memorable browsing experience.

## 🚀 Features

* 🎬 Spider-Verse inspired UI
* ⚡ Built with Next.js App Router
* 🎨 Modern Tailwind CSS design system
* ✨ Smooth Framer Motion animations
* 🖱️ Interactive drag-and-drop playground

## ⚡ Getting Started

Clone the repository

```bash
git clone <repository-url>
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Open

```text
http://localhost:3000
```
